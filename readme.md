# darkweb scraper

Foobar is a Python library for dealing with word pluralization.

## Installation

```bash
npm i
```

## Usage
clone this repo

put your configuration in ./configs/sites

```bash
cd src
npx prisma migrate reset
npm run scrap
```
```bash
npm run scrap
```
